# Messages — Fleet Coordination

This directory holds bottles and bulletins for the fleet. Anyone in the
Iron-to-Iron network can read, write, and respond. Messages here are
durable — they're committed to git and pushed to GitHub.

Convention:
- `BOTTLE-TO-{RECIPIENT}-{DATE}-{TOPIC}.md` — directed messages
- `FLEET-BULLETIN-{DATE}-{TOPIC}.md` — broadcast to everyone
- Reply by writing a BOTTLE-FROM-{YOU} in response
